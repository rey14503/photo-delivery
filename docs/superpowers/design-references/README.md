# Design References

Drop visual/prototype output from external tools (e.g. Google AI Studio's Build mode) here — one subfolder per source, e.g. `ai-studio-dashboard-mockup/`.

**These are reference material only, never production code.** Nothing in this folder is imported by the app. Whoever picks up the corresponding plan (see `docs/superpowers/specs/2026-07-09-ui-architecture-adr.md`) should look at what's here for layout/visual direction, then implement it as real components inside `src/`, following the existing codebase's conventions (component boundaries, prop shapes, TDD, error handling) — not by copy-pasting generated code wholesale, since tools like AI Studio don't know this app's data model, existing components, or established patterns.

When adding something here, note in this file: what it is, which spec/plan it's for, and the date.
